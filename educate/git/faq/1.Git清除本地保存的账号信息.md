Git清除本地保存的账号信息
===

<div class="jumbotron">
	<p>Git是一个很好的文件管理工具, Git的账号信息默认会存储在本地. 如果你有多个Git账号, 向不同的账号仓库提交文件时需要切换账号. 这时需要清除原有存储的账号信息, 否则会提示验证失败.
	</p>
</div>

① 清除步骤
---

`开始菜单`->`控制面板`->`凭据管理器`->`普通凭据`->`从保管库中删除`.
