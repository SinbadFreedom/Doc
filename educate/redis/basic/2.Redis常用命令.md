2.Redis常用命令
===

1. redis常用命令
---

###命令行连接Redis
	
	redis-cli

###查看所有的key

	keys *

<div class="bs-callout bs-callout-warning">
	<p>keys命令后边没有分号, 中间有空格</p>
</div>

###查看key的类型

	type BO:Server

###获取一个key

	get key

###删除key
删除单个key

	del key

删除多个key

	redis-cli -a pass(密码) keys "WX_ACT_USER_KEY_*" | xargs redis-cli -a pass(密码) del

###返回满足给定patterm的所有key

	keys    #(*)取出所有的key
			(my*)取出和关键字相关的
	
###确认一个key是否存在

	exists
	
###设置一个key的过期时间

	expire
	
###获取hash类型的全部数据

	hgetall key
	
###获取hash类型的一个数据

	hget key key1

###将当前数据库中的key转移到其他数据库中

	move            #0-15数据库
	
###移除给定key的过期时间

	persist                    #取消定时
	
###实时传储收到的请求
	
	config get dir/* 

###选择一个数据库

	select 0

`0`为数据库ID

###测试是否联通

	ping

###退出连接

	quit

###返回当前数据库中key的数量

	dbsize

###获取服务器信息

	info

###删除当前选择的数据库中的key

	flushdb
	
###删除所有数据库中的所有key

	flushall

###设置密码

	vi /etc/redis.conf
	requirepass pp123456
	pkill redis     #杀死进程
	service  redis start 
	auth        pp123456
