###Hello World
我们需要写一个最简单的Java程序.
新建一个文本文件`HelloWorld.java`，然后写入以下内容:    

```java
public class HelloWorld {
	public static void main(String[] args) {
		System.out.println("Hello world!");
	}
}
```
