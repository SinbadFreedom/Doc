也叫`条件运算符`.   
由两个符号组成`?`和`:`,要求有3个操作对象,所以也叫三目运算符.   

##表达式:

	a<b?a:b;

如果`?`前边的表达式(a<b)为`true`,取`:`前边的值,即a值.
如果`?`前边的表达式(a<b)为`false`,取`:`后边的值,即b值.

##示例代码
```java
package lession12;

/**
 * 大屎蛋教程网-dashidan.com
 *
 * 教程: 12.Java三目运算符： ？：
 */
public class Demo1 {
    public static void main(String[] args) {
        int a = 1;
        int b = 2;
        int c = a > b ? a : b;
        /** a > b 为false,所以取":"后边的值b赋值给c, 故c为2.*/
        System.out.println(c);
    }
}
```
##输出
	2
