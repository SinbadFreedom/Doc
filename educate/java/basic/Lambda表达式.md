Lambda表达式
===

如果接口只有1个方法, 这时使用匿名内部类时代码看上去不清晰也不简洁. 在这种情况下, 你会希望将函数式(functionality)作为参数传递进去, 来让代码看起来更加简洁. 例如: 处理按钮点击事件时.

Lambda表达式可以实现这种需求, 将函数式作为方法参数, 者将代码作为数据. 可以将单方法(single-method)类更加简洁.
