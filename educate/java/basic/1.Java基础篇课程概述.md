###java平台类型
Java有3种基础的开发包类型:   
`J2SE`: 标准版,应用最广的类型.包括Android开发和大量服务器后台开发基于这个平台.(推荐版)   
`J2EE`: 企业版,功能繁杂,新手可以忽略.   
`J2ME`: 几乎已经被淘汰的手机版.在Android智能机火爆之前曾经一统天下,如今已是明日黄花,新手可以忽略.   

###Java最新版本
目前最新版是`JDK8.0`.

###选择平台和版本
从J2SE,JDK8.0开始.
