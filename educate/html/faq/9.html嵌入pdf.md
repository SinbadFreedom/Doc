<p>HTML嵌入pdf文档常用的3种方法.</p>  
</div>

1 使用iframe嵌入pdf
===

```html
<iframe src="大屎蛋教程网.pdf" with="800" height="600" scroll="no"></iframe>
```

2 使用`<object>`标记
===

```html
<object width="800" height="600">
	<param name="src" value="大屎蛋教程网.pdf">
</object>
```

3 加入链接直接打开pdf
===

```html
<a href="大屎蛋教程网.pdf">大屎蛋教程网</a>
```
