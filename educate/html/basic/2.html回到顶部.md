html回到页面顶部的方式
===

1. 链接标签中加上`href="#top"`属性
---
这样点击这个标签时会返回顶部.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>html回到页面顶部</title>
</head>
<body>
<div style="width: 600px; height: 1500px; background-color: #4cae4c">
    <p>
        大屎蛋教程网-dashidan.com
    </p>
</div>
<div>
    <a href="#top">点击回顶部</a>
</div>
</body>
</html>
```
