ubuntu安装配置redis及常用命令
===

1 ubuntu安装redis服务
---

输入命令:

	apt-get install redis-server
	
2 ubuntu安装php对redis的支持
---

执行以下脚本
	apt-get install php5-redis
	
<div class="bs-callout bs-callout-warning">
	<p>这一步不是必须, 根据自己的需要安装.</p>
</div>	
	
3 启动和停止
---

	service redis start
	service redis stop
