Linux中文乱码
===

<div class="jumbotron">
<p>linux部分系统默认语言设置为英文, 会导致中文显示乱码.</p>  
</div>

① 查看系统编码   
---
```
echo $LANG
```

② 修改编码
---
```
export LANG=zh_CN.UTF-8
```

③ 修改/etc/environment
---
```
vim /etc/environment
```
	
文件末尾加入:
```
LANG="zh_CN:zh:en_US:en"
LANGUAGE="zh_CN:zh:en_US:en"
```
④ linux运行java程序,控制台中文乱码
---
启动时加入参数:
```
java -Dfile.encoding="UTF-8" -jar xxx.jar
```
