1 输入修改root密码命令
===

1.1 输入修改密码命令
---
修改root密码需要先切换为root用户.在linux的shell控制台输入命令:

```
sudo passwd root
```

命令参数说明:

- sudo 切换root用户
- passwd 修改密码
- root 需要修改密码的用户名

1.2 输入新密码:
---
在shell命令行输入新密码.


1.3 再次输入新密码
---
	
两次输入密码必须一致.完成后按回车确认.会出现提示`passwd: password updated successfully`表示修改密码成功.
