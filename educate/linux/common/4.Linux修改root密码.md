Linux修改root密码
===

<div class="jumbotron">
<p>想要一个称心如意的root密码, 看这个文章就够了.</p>  
</div>

① 输入命令
---

```
sudo passwd root
```

② Enter new UNIX password:
---
输入新密码:

```
xxxxxxxx
```

③ Retype new UNIX password:
---
再次输入新密码:

```
xxxxxxxx
```
	
提示`passwd: password updated successfully`修改成功.
