Ubuntu为apt-get

CentOS为yum

① 检查php是否支持redis

通过phpinfo()方法.将一下代码复制粘贴到info.php文件中.然后通过浏览器打开info.php文件会显示php的信息.

在显示内容中搜索redis, 能够找到说明支持, 找不到说明不支持.

<?php
phpinfo();
?>
② 下载phpredis
wget https://github.com/nicolasff/phpredis/zipball/master -O php-redis.zip
③ 解压
unzip php-redis.zip
④ 安装php5-dev
apt-get install php5-dev
⑤ 执行phpize

进入php-redis解压目录执行phpize

cd phpredis-phpredis-24e7859/
phpize5
⑥ 编译
./configure
make
make install
⑦ 生成redis.so文件
/usr/lib/php5/20090626/redis.so
⑧ 将这个文件加入php.ini中的extension

打开php.ini文件, 在末尾加入

extension=/usr/lib/php5/20090626/redis.so;
⑨ 重启php-fpm
service php5-fpm restart
⑩ 重新检查phpinfo

浏览器打开info.php, 搜索redis. 能找到说明成功.
