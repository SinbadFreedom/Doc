1 安装php5-dev依赖

安装php-redis扩展, 需要通过phpize5命令.这个命令在php5-dev包中.

apt-get install php5-dev

2 下载redis-3.1.4.tgz

wget https://github.com/nicolasff/phpredis/zipball/master -O php-redis.zip

3 解压

tar zxvf redis-3.1.4.tgz

4 进入解压目录redis-3.1.4

cd redis-3.1.4

5 运行phpize5

phpize5

6 编译运行

./configure
make
make install
