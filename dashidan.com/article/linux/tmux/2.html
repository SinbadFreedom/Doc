Ubuntu安装配置tmux及常用命令
① ubuntu安装tmux服务必需组件

输入命令:

apt-get install libevent-devel ncurses-devel
② 下载tmux源码安装

输入命令:

wget http://downloads.sourceforge.net/tmux/tmux-1.6.tar.gz
tar zxvf tmux-1.6.tar.gz
cd tmux-1.6
./configure
make
make install
