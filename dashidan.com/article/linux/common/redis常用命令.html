redis常用命令

1. 命令行连接Redis2. 查看所有的key3. 查看key的类型4. 获取一个key5. 删除key6. 返回满足给定patterm的所有key7. 确认一个key是否存在8. 设置一个key的过期时间9. 获取hash类型的全部数据10. 获取hash类型的一个数据11. 将当前数据库中的key转移到其他数据库中12. 移除给定key的过期时间13. 实时传储收到的请求14. 选择一个数据库15. 测试是否联通16. 退出连接17. 返回当前数据库中key的数量18. 获取服务器信息19. 删除当前选择的数据库中的key20. 删除所有数据库中的所有key21. 设置密码

1. 命令行连接Redis

redis-cli

2. 查看所有的key

keys *

keys命令后边没有分号, 中间有空格

3. 查看key的类型

type BO:Server

4. 获取一个key

get key

5. 删除key

删除单个key

del key

删除多个key

redis-cli -a pass(密码) keys "WX_ACT_USER_KEY_*" | xargs redis-cli -a pass(密码) del

6. 返回满足给定patterm的所有key

keys    #(*)取出所有的key
        (my*)取出和关键字相关的

7. 确认一个key是否存在

exists

8. 设置一个key的过期时间

expire

9. 获取hash类型的全部数据

hgetall key

10. 获取hash类型的一个数据

hget key key1

11. 将当前数据库中的key转移到其他数据库中

move            #0-15数据库

12. 移除给定key的过期时间

persist                    #取消定时

13. 实时传储收到的请求

config get dir/*

14. 选择一个数据库

select 0

0为数据库ID

15. 测试是否联通

ping

16. 退出连接

quit

17. 返回当前数据库中key的数量

dbsize

18. 获取服务器信息

info

19. 删除当前选择的数据库中的key

flushdb

20. 删除所有数据库中的所有key

flushall

21. 设置密码

vi /etc/redis.conf
requirepass pp123456
pkill redis     #杀死进程
service  redis start
auth        pp123456
