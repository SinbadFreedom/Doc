npm install -g与--save参数

1. 本地安装

2. 全局安装

3. --save参数

npm的包安装分为本地安装(local)、全局安装(global)两种, -g表示全局安装. --save表示是否加入`package.json`的`dependencies`中.

1. 本地安装

npm install不加-g参数, 为本地安装.
将安装包放在./node_modules下(运行npm时所在的目录).

2. 全局安装

npm install -g为全局安装.
安装nodejs的模板作为全局模块, 下载到系统默认的node_mudule目录中, 而不是项目中的node_mudole目录.

3. --save参数

使用npm install的--save参数, 将安装的包加入package.json的dependencies中.
以后在使用npm install会自动下载dependencies中的包.
