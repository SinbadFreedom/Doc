6 Java对象(5)：继承：extends

6.1 继承方式6.2 java单根继承6.3 示例代码6.4 相关文章

Java继承的本质是子类继承父类, 拥有父类的属性和方法, 是代码复用基础. 子类可以有自己的方法和属性.

6.1 继承方式

通过关键字extends. 我们以猫和狗继承是宠物, 宠物都有吃的行为. 猫有吃鱼的行为, 狗有吃骨头的行为为例, 实现这个继承关系.

父类Pet:

package com.dashidan.promote.lession6;

/**
 * 大屎蛋教程网-localhost
 *
 * Java教程进阶篇: 6.Java对象(5)：继承：extends
 * Pet类
 */
public class Pet {
    public void eat() {
        System.out.println("Pet eat");
    }
}

子类Cat:

package com.dashidan.promote.lession6;

/**
 * 大屎蛋教程网-localhost
 *
 * Java教程进阶篇: 6.Java对象(5)：继承：extends
 * Cat 继承自 Pet类
 */
public class Cat extends Pet {
    public void eatFish() {
        System.out.println("Cat eatFish.");
    }
}

子类Dog:

package com.dashidan.promote.lession6;

/**
 * 大屎蛋教程网-localhost
 *
 * Java教程进阶篇: 6.Java对象(5)：继承：extends
 * Dog 继承自 Pet类
 */
public class Dog extends Pet {
    public void eatBone() {
        System.out.println("Dog eatBone.");
    }
}

6.2 java单根继承

java单根继承是指单个类只能继承一个父类, 在c++中是多根继承, 一个类可以同时继承多个父类.

6.3 示例代码

package com.dashidan.promote.lession6;

/**
 * 大屎蛋教程网-localhost
 *
 * Java教程进阶篇: 6.Java对象(5)：继承：extends
 */
public class Demo1 {

    public static void main(String[] args) {
        Cat cat = new Cat();
        Dog dog = new Dog();
        /** 调用父类的方法*/
        cat.eat();
        dog.eat();
        /** 调用子类的方法*/
        cat.eatFish();
        dog.eatBone();
    }

}

输出:

Pet eat
Pet eat
Cat eatFish.
Dog eatBone.
