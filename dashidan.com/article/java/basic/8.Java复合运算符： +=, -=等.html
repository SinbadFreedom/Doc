8.Java复合运算符： +=, -=等

复合运算符是一种简写.以+=为例,表示加一个值再赋值给本身. 常用的复合运算符 += 加法赋值 -= 减法赋值 *= 乘法赋值 /= 除法赋值 %= 模运算赋值

例:

// += 加法赋值
a += 1;
// 等同于以下
a = a + 1;

//-= 减法赋值
a -= 1;
// 等同于以下
a = a - 1;

//*= 乘法赋值
a *= 1;
// 等同于以下
a = a * 1;

//*= 模运算赋值
a /= 1;
// 等同于以下
a = a / 1;

//%= 取余赋值
a %= 1;
// 等同于以下
a = a % 1;
