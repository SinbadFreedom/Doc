12.Java三目运算符： ？：

也叫条件运算符.
由两个符号组成?和:,要求有3个操作对象,所以也叫三目运算符.

表达式:

a<b?a:b;

详细解读:

如果`?`前边的表达式(a<b)为`true`,取`:`前边的值,即a值.
如果`?`前边的表达式(a<b)为`false`,取`:`后边的值,即b值.

int a = 1;
int b = 2;
int c = a > b ? a : b;
// a > b 为false,所以取":"后边的值b赋值给c, 故c为2.
System.out.println(c);
